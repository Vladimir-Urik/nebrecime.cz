<h2 id="zaklady">Základy JavaScriptu</h2>
<p><b>JavaScript</b> je interpretovaný, dynamicky typovaný jazyk běžící v prohlížeči (i na serveru přes Node.js). Připojení k HTML:</p>
<code><span class="comment">&lt;!-- Inline --&gt;</span>
<span class="b">&lt;script&gt;</span>console.log('Hello');<span class="b">&lt;/script&gt;</span>

<span class="comment">&lt;!-- Externí soubor (doporučeno, umístit před &lt;/body&gt;) --&gt;</span>
<span class="b">&lt;script</span> <span class="g">src</span>=<span class="r">"script.js"</span><span class="b">&gt;&lt;/script&gt;</span></code>

<h3>Proměnné</h3>
<code><span class="p">let</span> x = 10;        <span class="comment">// mění se</span>
<span class="p">const</span> PI = 3.14;   <span class="comment">// konstantní</span>
<span class="p">var</span> stary = true;  <span class="comment">// starý způsob, nepoužívat</span></code>

<h3>Podmínky a cykly</h3>
<code><span class="comment">// Podmínky</span>
<span class="p">if</span> (x > 0) {
  console.log(<span class="r">'kladné'</span>);
} <span class="p">else if</span> (x === 0) {
  console.log(<span class="r">'nula'</span>);
} <span class="p">else</span> {
  console.log(<span class="r">'záporné'</span>);
}

<span class="comment">// Cykly</span>
<span class="p">for</span> (<span class="p">let</span> i = 0; i < 5; i++) { ... }
<span class="p">while</span> (podminka) { ... }

<span class="comment">// Pole – for...of</span>
<span class="p">for</span> (<span class="p">const</span> item <span class="p">of</span> pole) { ... }</code>

<h2 id="datove-typy">Datové typy a pole</h2>
<table>
  <tr><th>Typ</th><th>Příklad</th></tr>
  <tr><td>number</td><td><code class="code">42</code>, <code class="code">3.14</code></td></tr>
  <tr><td>string</td><td><code class="code">"ahoj"</code>, <code class="code">`šablona ${x}`</code></td></tr>
  <tr><td>boolean</td><td><code class="code">true</code>, <code class="code">false</code></td></tr>
  <tr><td>null / undefined</td><td>prázdná / neinicializovaná hodnota</td></tr>
  <tr><td>object</td><td><code class="code">{ jmeno: "Jan", vek: 20 }</code></td></tr>
  <tr><td>array</td><td><code class="code">[1, 2, 3]</code></td></tr>
</table>

<h3>Práce s polem</h3>
<code><span class="p">const</span> arr = [<span class="r">3, 1, 4, 1, 5</span>];
arr.push(<span class="r">9</span>);              <span class="comment">// přidá na konec</span>
arr.pop();               <span class="comment">// odebere z konce</span>
arr.sort((a, b) => a - b); <span class="comment">// vzestupné řazení</span>
arr.filter(x => x > 3);  <span class="comment">// [4, 5, 9]</span>
arr.map(x => x * 2);     <span class="comment">// [6, 2, 8, ...]</span></code>

<h2 id="funkce">Funkce</h2>
<code><span class="comment">// Klasická funkce</span>
<span class="p">function</span> secti(a, b) {
  <span class="p">return</span> a + b;
}

<span class="comment">// Anonymní funkce</span>
<span class="p">const</span> nasob = <span class="p">function</span>(a, b) { <span class="p">return</span> a * b; };

<span class="comment">// Arrow funkce</span>
<span class="p">const</span> umocni = (x) => x ** 2;

<span class="comment">// Callback</span>
setTimeout(() => console.log(<span class="r">'hotovo'</span>), 1000);

[<span class="r">1,2,3</span>].forEach(n => console.log(n)); <span class="comment">// callback</span></code>

<h2 id="dom">Manipulace s DOM</h2>
<p>DOM (Document Object Model) je stromová reprezentace HTML dostupná přes JS.</p>
<code><span class="comment">// Výběr elementů</span>
<span class="p">const</span> el = document.getElementById(<span class="r">'mujDiv'</span>);
<span class="p">const</span> els = document.querySelectorAll(<span class="r">'.trida'</span>);

<span class="comment">// Změna obsahu a stylu</span>
el.textContent = <span class="r">'Nový text'</span>;
el.classList.add(<span class="r">'aktivni'</span>);
el.style.color = <span class="r">'red'</span>;

<span class="comment">// Události</span>
el.addEventListener(<span class="r">'click'</span>, () => {
  alert(<span class="r">'kliknuto!'</span>);
});

<span class="comment">// Časovače</span>
setTimeout(fn, 1000);    <span class="comment">// jednou za 1s</span>
setInterval(fn, 1000);   <span class="comment">// opakovaně každou 1s</span></code>

<h2 id="async">JSON a async/await</h2>
<h3>JSON</h3>
<p>Textový formát pro výměnu dat (JavaScript Object Notation).</p>
<code><span class="p">const</span> json = <span class="r">'{"jmeno":"Jan","vek":20}'</span>;
<span class="p">const</span> obj = JSON.parse(json);   <span class="comment">// string → objekt</span>
<span class="p">const</span> str = JSON.stringify(obj); <span class="comment">// objekt → string</span></code>

<h3>Fetch API (async/await)</h3>
<code><span class="p">async function</span> nactiData() {
  <span class="p">const</span> res = <span class="p">await</span> fetch(<span class="r">'https://api.example.com/data'</span>);
  <span class="p">const</span> data = <span class="p">await</span> res.json();
  console.log(data);
}</code>

<h2 id="canvas">Canvas a Node.js</h2>
<h3>Kreslení ve 2D canvasu</h3>
<code><span class="b">&lt;canvas</span> <span class="g">id</span>=<span class="r">"platno"</span> <span class="g">width</span>=<span class="r">"400"</span> <span class="g">height</span>=<span class="r">"300"</span><span class="b">&gt;&lt;/canvas&gt;</span></code>
<code><span class="p">const</span> canvas = document.getElementById(<span class="r">'platno'</span>);
<span class="p">const</span> ctx = canvas.getContext(<span class="r">'2d'</span>);

ctx.fillStyle = <span class="r">'blue'</span>;
ctx.fillRect(<span class="r">10, 10, 100, 50</span>);  <span class="comment">// obdélník</span>

ctx.beginPath();
ctx.arc(<span class="r">200, 150, 50, 0, Math.PI * 2</span>);  <span class="comment">// kruh</span>
ctx.fill();</code>

<h3>Node.js</h3>
<p>Prostředí pro spouštění JavaScriptu na serveru (mimo prohlížeč). Umožňuje psát webové servery, CLI nástroje a backendy v JS. Základem je událostní smyčka (event loop) a balíčkový manažer <b>npm</b>.</p>
