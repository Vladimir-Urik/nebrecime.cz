<h2 id="html-zaklady">HTML základy</h2>
<p><b>HTML</b> (HyperText Markup Language) je značkovací jazyk pro tvorbu webových stránek. Definuje strukturu a obsah stránky pomocí elementů — tagů.</p>

<h3>Struktura HTML dokumentu</h3>
<code><span class="b">&lt;!DOCTYPE html&gt;</span>
<span class="b">&lt;html</span> <span class="g">lang</span>=<span class="r">"cs"</span><span class="b">&gt;</span>
<span class="b">&lt;head&gt;</span>
  <span class="b">&lt;meta</span> <span class="g">charset</span>=<span class="r">"UTF-8"</span><span class="b">&gt;</span>
  <span class="b">&lt;title&gt;</span>Název stránky<span class="b">&lt;/title&gt;</span>
<span class="b">&lt;/head&gt;</span>
<span class="b">&lt;body&gt;</span>
  <span class="comment">&lt;!-- obsah stránky --&gt;</span>
<span class="b">&lt;/body&gt;</span>
<span class="b">&lt;/html&gt;</span></code>

<ul>
  <li><b>&lt;!DOCTYPE html&gt;</b> — deklarace verze HTML (HTML5)</li>
  <li><b>&lt;html&gt;</b> — kořenový element celého dokumentu</li>
  <li><b>&lt;head&gt;</b> — metadata stránky (název, CSS, charset…)</li>
  <li><b>&lt;body&gt;</b> — viditelný obsah stránky</li>
</ul>

<h2 id="css-zaklady">CSS základy</h2>
<p><b>CSS</b> (Cascading Style Sheets) definuje vzhled HTML elementů — barvy, fonty, rozložení, animace.</p>

<h3>Syntaxe</h3>
<code><span class="p">selektor</span> {
  <span class="g">vlastnost</span>: <span class="r">hodnota</span>;
  <span class="g">vlastnost2</span>: <span class="r">hodnota2</span>;
}</code>

<h3>Způsoby připojení CSS</h3>
<ul>
  <li><b>Externí soubor</b> — <code class="code"><span class="b">&lt;link</span> <span class="g">rel</span>=<span class="r">"stylesheet"</span> <span class="g">href</span>=<span class="r">"style.css"</span><span class="b">&gt;</span></code> (doporučeno)</li>
  <li><b>Interní</b> — <code class="code"><span class="b">&lt;style&gt;</span></code> v <code class="code">&lt;head&gt;</code></li>
  <li><b>Inline</b> — atribut <code class="code"><span class="g">style</span>=<span class="r">"..."</span></code> přímo na elementu</li>
</ul>

<h3>Specifičnost (priorita pravidel)</h3>
<p>Čím specifičtější selektor, tím vyšší priorita: <b>ID &gt; třída &gt; element</b>. Klíčové slovo <code class="code"><span class="r">!important</span></code> přebíjí vše (používat střídmě).</p>

<h3>Google Fonts</h3>
<p>Webové fonty se připojují přes <code class="code"><span class="b">&lt;link&gt;</span></code> v <code class="code">&lt;head&gt;</code>:</p>
<code><span class="b">&lt;link</span> <span class="g">rel</span>=<span class="r">"preconnect"</span> <span class="g">href</span>=<span class="r">"https://fonts.googleapis.com"</span><span class="b">&gt;</span>
<span class="b">&lt;link</span> <span class="g">href</span>=<span class="r">"https://fonts.googleapis.com/css2?family=Roboto&amp;display=swap"</span> <span class="g">rel</span>=<span class="r">"stylesheet"</span><span class="b">&gt;</span></code>

  <span class="g">font-family</span>: <span class="r">"Roboto", sans-serif</span>;
}</code>
<p><b>Bezpečné fonty</b> (dostupné bez importu): Arial, Verdana, Georgia, Times New Roman, Courier New, Trebuchet MS.</p>

<h2 id="pozicovani">Pozicování</h2>
<p>Vlastnost <code class="code"><span class="g">position</span></code> určuje, jak je element umístěn v dokumentu.</p>

<h3>Typy pozicování</h3>
<table>
  <tr><th>Hodnota</th><th>Popis</th></tr>
  <tr><td><code class="code"><span class="r">static</span></code></td><td>Výchozí — element v normálním toku</td></tr>
  <tr><td><code class="code"><span class="r">relative</span></code></td><td>Posunutí od původní pozice (prostor zůstává)</td></tr>
  <tr><td><code class="code"><span class="r">absolute</span></code></td><td>Vyřazení z toku, pozice vůči nejbližšímu positioned rodiči</td></tr>
  <tr><td><code class="code"><span class="r">fixed</span></code></td><td>Fixní vůči viewportu (zůstane při scrollování)</td></tr>
  <tr><td><code class="code"><span class="r">sticky</span></code></td><td>Kombinace relative + fixed (přilepí se při scrollu)</td></tr>
</table>

<code><span class="p">.rodic</span> {
  <span class="g">position</span>: <span class="r">relative</span>; <span class="comment">/* kotva pro absolutně pozicované dítě */</span>
}

<span class="p">.dite</span> {
  <span class="g">position</span>: <span class="r">absolute</span>;
  <span class="g">top</span>: <span class="r">0</span>;
  <span class="g">right</span>: <span class="r">0</span>;
}

<span class="p">.hlavicka</span> {
  <span class="g">position</span>: <span class="r">sticky</span>;
  <span class="g">top</span>: <span class="r">0</span>; <span class="comment">/* přilepí se k hornímu okraji při scrollování */</span>
}</code>

<h3>Z-index</h3>
<p><code class="code"><span class="g">z-index</span></code> určuje pořadí vrstvení elementů (funguje pouze u <b>positioned</b> elementů).</p>
<code><span class="p">.modal</span> {
  <span class="g">position</span>: <span class="r">fixed</span>;
  <span class="g">z-index</span>: <span class="r">1000</span>; <span class="comment">/* vyšší číslo = nahoře */</span>
}</code>

<h3>Speciální znaky a čeština</h3>
<p>Správné nastavení pro česky psané stránky:</p>
<code><span class="b">&lt;meta</span> <span class="g">charset</span>=<span class="r">"UTF-8"</span><span class="b">&gt;</span>
<span class="b">&lt;html</span> <span class="g">lang</span>=<span class="r">"cs"</span><span class="b">&gt;</span></code>
<p>Speciální znaky v HTML se zapisují HTML entitami: <code class="code">&amp;amp;</code> → &amp;, <code class="code">&amp;lt;</code> → &lt;, <code class="code">&amp;gt;</code> → &gt;, <code class="code">&amp;nbsp;</code> → nezlomitelná mezera.</p>

<h2 id="box-model">Box model</h2>
<p>Každý HTML element je obdélníkový box skládající se z:</p>
<ul>
  <li><b>content</b> — samotný obsah (text, obrázek)</li>
  <li><b>padding</b> — vnitřní odsazení (mezi obsahem a okrajem)</li>
  <li><b>border</b> — rámeček okolo elementu</li>
  <li><b>margin</b> — vnější odsazení (prostor mezi elementy)</li>
</ul>
<code><span class="p">.box</span> {
  <span class="g">width</span>: <span class="r">200px</span>;
  <span class="g">padding</span>: <span class="r">16px</span>;
  <span class="g">border</span>: <span class="r">2px solid #333</span>;
  <span class="g">margin</span>: <span class="r">24px</span>;
  <span class="g">box-sizing</span>: <span class="r">border-box</span>; <span class="comment">/* padding a border se počítají do width */</span>
}</code>

<h2 id="layout">Layout</h2>

<h3>Flexbox</h3>
<p>Jednorozměrný layout (řádek nebo sloupec).</p>
<code><span class="p">.container</span> {
  <span class="g">display</span>: <span class="r">flex</span>;
  <span class="g">justify-content</span>: <span class="r">space-between</span>; <span class="comment">/* hlavní osa */</span>
  <span class="g">align-items</span>: <span class="r">center</span>;          <span class="comment">/* příčná osa */</span>
  <span class="g">gap</span>: <span class="r">16px</span>;
}</code>

<h3>Grid</h3>
<p>Dvourozměrný layout (řádky + sloupce).</p>
<code><span class="p">.grid</span> {
  <span class="g">display</span>: <span class="r">grid</span>;
  <span class="g">grid-template-columns</span>: <span class="r">1fr 1fr 1fr</span>; <span class="comment">/* 3 stejné sloupce */</span>
  <span class="g">gap</span>: <span class="r">24px</span>;
}</code>

<h3>Responzivní design — Media Queries</h3>
<code><span class="comment">/* Výchozí styl pro desktop */</span>
<span class="p">.container</span> { <span class="g">width</span>: <span class="r">1200px</span>; }

<span class="comment">/* Pro obrazovky do 768px (mobil) */</span>
<span class="b">@media</span> (<span class="g">max-width</span>: <span class="r">768px</span>) {
  <span class="p">.container</span> { <span class="g">width</span>: <span class="r">100%</span>; }
}</code>
